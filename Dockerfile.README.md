# rfc-bot

## A GitHub bot built with Probot aiming to help in the creation of RFC proposal referenda.

To run the bot via Docker:

```
$ docker run \
    -e APP_ID=<app-id> \
    -e PRIVATE_KEY=<pem-value> \
    paritytech/rfc-bot
```
